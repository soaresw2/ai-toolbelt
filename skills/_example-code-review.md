# Code Review Skill

This skill enables the AI to perform thorough code reviews.

## Capabilities

- Analyze code for potential bugs and issues
- Check code style and formatting consistency
- Identify security vulnerabilities
- Suggest performance improvements
- Verify adherence to best practices and design patterns
- Review test coverage
- Check for proper error handling
- Suggest code documentation improvements

## Usage

When activated, this skill will:
1. Analyze the provided code systematically
2. Categorize findings by severity (critical, major, minor)
3. Provide specific, actionable feedback
4. Suggest concrete improvements with code examples
5. Highlight positive aspects of the code

## Best Practices

- Focus on constructive feedback
- Prioritize critical issues over style preferences
- Consider the project context and existing patterns
- Balance thoroughness with practicality
